Gregotron Privacy Policy

Info

Here is stated what data greg will collect and use.

Types of data collected

Some data greg may use include: Messages - These messages are logged for moderation purposes and are deleted within a few hours to a few days. Moderated messages may also be logged in a logs channel on the discord server where the message may of been moderated Usernames and User IDs - Gregotron collects this information for the Gregotron RPG system these are required to use the system We do not save any personal information

More info

Info saved by greg gets deleted after a few hours to a few days we do not save messages forever as this would breach Discord TOS. The reason for this info being saved in the first place is for specific commands to be able to function we do not log the entire chat only moderated messages. We will not share or sell your data.
